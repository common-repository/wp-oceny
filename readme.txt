== Description ==

Wtyczka umożliwiająca odwiedzającym ocenianie postów.

== Installation ==

1. Skopiuj wszystkie pliki do katalogu **/wp-content/plugins/wp-oceny/**
2. Aktywuj plugin w panelu administracyjnym
3. Skonfiruruj ustawienia pluginu w zakładce **Ustawienia/WP Oceny**

== Frequently Asked Questions ==

= Czy można zastąpić domyslne grafiki zastosowane we wtyczce? =
Oczywiście. Ważne jest tylko, by wprowadzić odpowiednie korekty w pliku **oceny.css** po zmianie grafik głosów na grafiki o innym rozmiarze.
